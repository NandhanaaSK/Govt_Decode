# 📄 Govt Decode

### Government Circular Summarizer

Govt Decode is an AI-powered web application that transforms lengthy government circulars, notifications, and policy documents into concise, easy-to-understand summaries. It helps citizens, students, professionals, and administrators quickly grasp key information without reading entire documents.

---

## ✨ Features

* 📤 Upload government circulars in PDF format
* 📝 Paste circular text directly into the application
* 🤖 AI-powered summarization using NLP and Transformer models
* 📏 Multiple summary lengths (Short, Medium, Detailed)
* 🏷️ Automatic category detection:

  * 🎓 Education
  * 🏥 Health
  * 💰 Finance
  * ⚖️ Legal
* 🔊 Text-to-Speech support for listening to summaries
* 📱 User-friendly and responsive interface
* 📰 Government updates and category-based information dashboard

## 🚀 How It Works

1. Upload a government circular PDF or paste text.
2. The document is cleaned and processed.
3. The AI model generates a concise summary.
4. The system automatically detects the document category.
5. Results are displayed as easy-to-read bullet points.
6. Users can listen to summaries using Text-to-Speech.
